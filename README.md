# Stock_Verse

A modern financial learning platform that gamifies stock market education for young investors, combining real-time market data with interactive games and smart assistance.

## Features

- Interactive Financial Dashboard
- Live Market Graphs
- Smart Investment Assistant
- Stock Market Games
- Financial Education Quizzes
- Leaderboard System
- Portfolio Management
- News Integration

## Technologies Used

### Frontend
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Chart.js for data visualization

### Backend
- Python
- Flask
- Advanced data processing

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Python 3.11+
- pip

### Frontend Setup
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend Setup
```bash
cd backend

# Install Python dependencies
python install_dependencies.py

# Start the server
python start_all_servers.py

# Setup the database
Nifty 50 past data to be inserted in a "data.csv" file located in the /backend directory
``` 
## Project Structure
- `/src` - Frontend React application
- `/backend` - Python backend services
- `/src/components` - React components
- `/src/data` - JSON data files
- `/src/services` - API services
